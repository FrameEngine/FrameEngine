\doxysubsection{Detailed Description}
Defines the \doxylink{classEntity}{Entity} class, which serves as a wrapper for ECS entities. 

The \doxylink{classEntity}{Entity} class provides a high-\/level abstraction over the Entity-\/\+Component-\/\+System (ECS), allowing entities to be created and manipulated without directly interfacing with the \doxylink{classRegistry}{Registry}.
