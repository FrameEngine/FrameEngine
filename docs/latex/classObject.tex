\doxysubsection{Detailed Description}
A high-\/level ECS entity wrapper with default components. 

The \doxylink{classObject}{Object} class extends \doxylink{classEntity}{Entity} to provide a ready-\/to-\/use object with predefined components.

Renders the object using the current shader from the renderer.

This method updates the model matrix and draws the object\textquotesingle{}s mesh.

This method retrieves the active shader from the provided \doxylink{classRenderer}{Renderer} instance, computes the model transformation matrix, and updates the shader uniforms (object color and model matrix). If the object is flagged for wireframe rendering, the Open\+GL polygon mode is temporarily set to GL\+\_\+\+LINE before drawing the mesh, then restored to GL\+\_\+\+FILL afterward.
