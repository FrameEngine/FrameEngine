\doxysubsection{Detailed Description}
An ECS entity representing a point light source. 

\doxylink{classPointLight}{Point\+Light} extends \doxylink{classObject}{Object}, automatically inheriting a \doxylink{structTransformComponent}{Transform\+Component} to manage its spatial position. However, it does not have a mesh associated with it.
