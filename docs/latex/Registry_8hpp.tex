\doxysubsection{Detailed Description}
Defines the \doxylink{classRegistry}{Registry} class for managing ECS entities and components. 

The \textquotesingle{}\doxylink{classRegistry}{Registry}\textquotesingle{} class is the core of the ECS (Entity-\/\+Component-\/\+System). It manages\+:
\begin{DoxyItemize}
\item \doxylink{classEntity}{Entity} creation and destruction.
\item Component storage and retrieval for each entity.
\end{DoxyItemize}

It uses a bitset for component tracking, allowing O(1) filtering of entities with specific components, as it can be critical for high-\/performance simulations.
