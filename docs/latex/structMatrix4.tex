\doxysubsection{Detailed Description}
A 4x4 matrix class. 

It provides standard arithmetic operations, matrix multiplication, and methods for generating specific transformation matrices.

Matrices are row-\/major ordered.

All angles are provided in degrees, unless specified otherwise.
