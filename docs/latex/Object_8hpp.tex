\doxysubsection{Detailed Description}
Defines the \doxylink{classObject}{Object} class, a special ECS entity wrapper. 

Every \doxylink{classObject}{Object} {\bfseries{automatically}} includes\+:
\begin{DoxyItemize}
\item \doxylink{structTransformComponent}{Transform\+Component} (for position, rotation, and scale).
\item Mesh\+Component (to store and manage the object\textquotesingle{}s geometry).
\end{DoxyItemize}

This makes \doxylink{classObject}{Object} good for representing planets, vehicles or any other dynamic renderable objects within the simulation.

\begin{DoxyNote}{Note}
If an entity does not require rendering, consider using a base \doxylink{classEntity}{Entity} instead. 
\end{DoxyNote}
