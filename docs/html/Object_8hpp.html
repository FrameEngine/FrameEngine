Detailed Description

Defines the Object class, a special ECS entity wrapper.

Every Object automatically includes:

  • TransformComponent (for position, rotation, and scale).
  • MeshComponent (to store and manage the object's geometry).

This makes Object good for representing planets, vehicles or any other dynamic renderable objects within the simulation.

Note
If an entity does not require rendering, consider using a base Entity instead.
