◆ fromMatrix()

Quaternion Quaternion::fromMatrix ( const Matrix4 & matrix)
static

Constructs a quaternion from a given rotation matrix.

This function uses the standard method of extracting a quaternion from a rotation matrix. It first computes the trace of the matrix to determine which case to use for extraction. The resulting quaternion is normalized before being returned.

TLDR; used formulaes from: https://www.euclideanspace.com/maths/geometry/rotations/conversions/matrixToQuaternion/index.htm

Parameters
matrixThe 4x4 rotation matrix to convert.
Returns
A normalized Quaternion representing the same rotation.
