Detailed Description

Defines the PhysicsSystem.

The PhysicsSystem processes all entities with 'PhysicsComponent' (stores velocity, acceleration, and mass).

It applies acceleration to velocity, then integrates velocity into position. This system should be called every frame with a fixed timestamp to maintain stable physics updates.

In the future will extend it with collision checks and other physics stuff
