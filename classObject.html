Detailed Description

A high-level ECS entity wrapper with default components.

The Object class extends Entity to provide a ready-to-use object with predefined components.

void Object::lookAt ( const Vector3 &  target)
inline

Orients the object so that it "looks at" a target position.

This method updates the object's rotation so that its forward direction points toward the target. It uses a look-at algorithm to compute a rotation matrix from the current position to the target, which is then converted into a quaternion and applied to the transform.

Parameters
targetThe target position to look at.

◆ render()

void Object::render ( Renderer &  renderer)

Renders the object.

Renders the object using the current shader from the renderer.

This method updates the model matrix and draws the object's mesh.

Parameters
rendererThe Renderer used to draw the object.

This method retrieves the active shader from the provided Renderer instance, computes the model transformation matrix, and updates the shader uniforms (object color and model matrix). If the object is flagged for wireframe rendering, the OpenGL polygon mode is temporarily set to GL_LINE before drawing the mesh, then restored to GL_FILL afterward.

Parameters
rendererThe Renderer instance responsible for drawing.
