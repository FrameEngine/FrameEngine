static

Creates a plane mesh.

The plane is centered at the origin and spans the given width and height. UV coordinates are generated by mapping the plane's X and Z coordinates to the [0, 1] range.

For each vertex: u = (posX + halfWidth) / width v = (posZ + halfHeight) / height

Parameters
widthThe total width of the plane.
heightThe total height of the plane.
segmentsThe number of subdivisions per edge.
Returns
A pointer to a new Mesh object.

unsigned int  latitudeSegments = 16

static

Creates a sphere mesh with spherical UV mapping.

UV mapping for a sphere is computed using spherical coordinates. For each vertex: u = phi / (2*PI) where phi is the azimuth angle (0 to 2*PI) v = theta / PI where theta is the polar angle (0 to PI)

Parameters
longitudeSegmentsNumber of segments along the longitude.
latitudeSegmentsNumber of segments along the latitude.
Returns
A pointer to a new Mesh object.
