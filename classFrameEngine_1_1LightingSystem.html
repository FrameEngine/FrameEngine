static void FrameEngine::LightingSystem::update ( )
inlinestatic

Updates the lighting UBO with data from all entities that have a LightComponent.

This function queries the global Registry for entities with both LightComponent and TransformComponent, fills the uboData structure accordingly, and uploads it to the GPU.
