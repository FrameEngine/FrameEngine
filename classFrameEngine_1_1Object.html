Detailed Description

A high-level ECS entity wrapper with default components.

The Object class extends Entity to provide a ready-to-use object with predefined components.

void FrameEngine::Object::lookAt ( const Vector3 &  target)
inline

Orients the object so that it "looks at" a target position.

This method updates the object's rotation so that its forward direction points toward the target. It uses a look-at algorithm to compute a rotation matrix from the current position to the target, which is then converted into a quaternion and applied to the transform.

Parameters
targetThe target position to look at.
