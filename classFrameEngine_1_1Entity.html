Detailed Description

A high-level ECS entity wrapper.

This class simplifies entity management by automatically handling unique IDs and component operations. It also enforces type safety while working with components.

Entities in this system are lightweight; they primarily serve as IDs that reference component data stored in the Registry.

Note
This design ensures that entities are just identifiers (not heavy objects), keeping performance optimal in large-scale simulations.
